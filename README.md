# Directus Application Setup

This project sets up a local Directus instance using Docker.

## Prerequisites

- Docker
- Docker Compose
- Git

## 1. Clone or Download the Project

```
git clone https://github.com/kavishka-wilagedara/directus-application.git
cd directus-assessment
```

## 2. Start the Services
```
docker-compose up -d
```
## 3. Access Directus
```
http://localhost:8055
```
